# 作業ログ (Walkthrough)
更新日: 2026-01-16
バージョン: dev1.0.0.20260116.1

## 本日の修正まとめ
1. **マスタデータの外部化**
   - 設定用シート `config` を1枚に集約。
   - アプリ起動時に1回だけ取得し、以降は再読込しないことで高速化。
2. **UI/UXの改善**
   - ヘッダーを「鍵取替管理」に変更。
   - 鍵情報（玄関新/古、物置、PS）をカード一覧と詳細画面の両方に配置。
   - 詳細画面の鍵情報を「太字・黒・大サイズ」にCSS調整し、視認性を最大化。
3. **入力制御の強化**
   - `onEntranceChange` メソッドにより「玄関新品」と「玄関中古」の二重選択を防止。
   - `saveEdit` メソッドで、進行中タブの時のみ担当者選択を必須バリデーション。
4. **表示の最適化**
   - 完了タブでは「完了日」を表示し、新規/進行中タブでは「予定日」を表示する出し分けを実装。
