# 鍵取替管理アプリ - グループ別アクセスガイド

このアプリは **依頼グループ** と **作業グループ** で異なる機能が利用できます。

---

## アクセスURL

| グループ | URLパラメータ | アクセス例 |
|----------|---------------|-----------|
| 依頼グループ | なし または `?mode=request` | `https://script.google.com/.../exec` |
| 作業グループ | `?mode=worker` | `https://script.google.com/.../exec?mode=worker` |

> **注意**: URLの末尾に `?mode=worker` を付けると作業グループ用画面が表示されます。

---

## 機能比較

| 機能 | 依頼グループ | 作業グループ |
|------|:------------:|:------------:|
| 新規依頼の追加（＋ボタン） | ✅ | ❌ |
| 基本情報の編集（編集ボタン） | ✅ | ❌ |
| 写真の撮り直し | ✅ | ❌ |
| 本日の更新内容の変更 | ✅ | ❌（読み取り専用） |
| 進捗更新（MMS受取日・受取担当者） | ❌ | ✅ |
| 完了報告 | ❌ | ✅ |
| 一覧の閲覧 | ✅ | ✅ |

---

## 依頼グループの使い方

依頼グループは、新規案件の登録と基本情報の管理を担当します。

### できること
- **新規案件登録**: 画面右下の「＋」ボタンから新規案件を登録
- **基本情報編集**: カードをタップ → 「内容を編集」ボタン → 地区・住所・号室・期限日などを変更
- **写真の撮り直し**: 編集モードで写真を再撮影
- **本日の更新内容**: ステータスを「更新作業中」「作業なし」「新規あり」から選択

---

## 作業グループの使い方

作業グループは、案件の進捗更新と完了報告を担当します。

### できること
- **進捗更新（新規タブ）**: カードをタップ → MMS受取日と受取担当者を入力 → 「進捗を更新」
- **完了報告（進行中タブ）**: カードをタップ → 完了報告者と完了日を入力 → 「完了報告」
- **本日の更新内容**: 依頼グループが設定した値を確認（変更不可）

```
     │                        │
     ├─ 新規案件登録 ─────────→│

     │←────────── MMS受取報告 ─┤
     │                        │

     │                        │
     ├─ ステータス更新 ────────→│（読み取り専用で確認）
     │                        │
```

---

## 簡易パスについて

各グループで異なる簡易パスを設定することで、アクセス制御が可能です。

- 依頼グループ用URL（パラメータなし）を依頼担当者に共有
- 作業グループ用URL（`?mode=worker`付き）を作業担当者に共有
